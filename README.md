# 🖋️ M Saad Iqbal | Digital Architect & Full-Stack Developer

A premium, editorial-first portfolio built for high-fidelity performance and aesthetic excellence. This project features a React frontend, an Express.js backend, and a robust Oracle Database integration.

---

## 🚀 Technical Highlights

- **Frontend**: React 19 with Vite, Framer Motion for liquid animations, and Tailwind CSS 4.
- **Backend**: Express.js with Firebase Auth for secure administrative access.
- **Database**: Oracle DB (SQL) for persistent project and profile management.
- **Storage**: Cloudinary integration for resilient media hosting.
- **Identity**: Full Dynamic CMS for managing biographical data, CV links, and studio identity.

---

## 🛠️ Project Structure

```bash
├── frontend    # React client-side application
├── backend     # Express.js API server
└── database    # SQL schemas and migration scripts
```

---

## ⚙️ Setup & Installation

### 1. Prerequisite
- Node.js (v18+)
- Oracle Database (Instant Client or Local Express Edition)
- Firebase Account (for Auth)
- Cloudinary Account (for Image Storage)

### 2. Backend Setup
1. Navigate to `/backend`.
2. Create a `.env` file based on `.env.example`.
3. Install dependencies: `npm install`.
4. Start server: `npm run dev`.

### 3. Frontend Setup
1. Navigate to `/frontend`.
2. Create a `.env` file based on `.env.example`.
3. Install dependencies: `npm install`.
4. Start development server: `npm run dev`.

---

## 🔒 Security
Sensitive credentials (DB passwords, Firebase keys, SMTP secrets) are strictly excluded via `.gitignore`. Refer to the `.env.example` files in each subdirectory for the required environment configuration.

## 🎨 Design Philosophy
The portfolio utilizes a **Dark Editorial** aesthetic, focusing on typography-led layouts and subtle glassmorphic textures. It is designed to feel like a high-end digital publication rather than a standard portfolio template.
